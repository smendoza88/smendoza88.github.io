C# (C-Sharp)

C# was created by Microsoft and released in July 2000, designed by Anders Hejlsberg as the flagship language of the new .NET Framework. It was built largely as Microsoft's answer to Java, which was rapidly winning over enterprise developers with its cross-platform promise. Microsoft needed a modern, type-safe, object-oriented language of its own — one that combined the power of C++ with the productivity of Java, while integrating tightly with Windows.

From Windows-Only to Cross-Platform

For its first decade, C# ran exclusively on Windows through the .NET Framework. That changed in 2016 when Microsoft released .NET Core, a complete rewrite of .NET that was open-source, modular, and able to run on Windows, Linux, and macOS. For the first time, C# code could be compiled and executed natively on Linux servers and containers.

In 2020, Microsoft unified everything under a single platform simply called .NET (starting with .NET 5), dropping the "Core" branding. Today, C# is a fully cross-platform language used to build web apps, cloud services, mobile apps, and games on virtually any operating system.

Key Features

  • Object-oriented and type-safe
  • Automatic memory management
  • Modern features like LINQ, async/await, and generics
  • Cross-platform via the unified .NET runtime

Common Uses

  • Web development (ASP.NET Core)
  • Cloud-based applications (Azure)
  • Game development (Unity)
  • Cross-platform desktop and mobile apps (.NET MAUI)
